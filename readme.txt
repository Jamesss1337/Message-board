db.php
    Файл отвечает за подключение к базе данных.

index.php
    Файл отвечает за отображение списка сообщений.

add.php
    Файл отвечает за добавление новых сообщений.

view.php
    Файл предназначен для просмотра отдельного сообщения и его комментариев.

edit.php
    Файл отвечает за редактирование существующих сообщений. 

add_comment.php
    Файл отвечает за добавление комментариев к сообщениям.
